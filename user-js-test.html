Canned Responses

This is a page for searching canned responses and other guidance for accessibility false positive reviews and alfa/platform consulting and support. The source data is a text file, and that text file is here.

Search

Info on how to search
  • You can specify multiple search terms, separated by spaces.
  • This page does an "and" search, not an "or" search.
  • This page does a substring search, not a whole word search.
  • This page generally expects each search term to be a fixed string. But a search term can be a regex instead - probably. This is not well defined.
  • The order of your search terms doesn't matter.
  • The search results are not sorted by "relevance" or a "score" or anything like that.
