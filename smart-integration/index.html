SMART browser extension

Instructions

One-time installation

  • Open Chrome
  • Go to chrome://extensions/
  • Find the "developer mode" toggle. (It's in the upper right corner, at the time of writing.) Enable this toggle.

Installation: on every new version

This means: follow these steps the first time you install this browser extension, and also every time you want to install a new version of it.
  • Download the zip file
  • Extract it to somewhere. For example, to a folder named C:\Users\YOUR_NAME\smart-browser-extension-v1.2
  • In Chrome, go to chrome://extensions/
  • Press the "load unpacked" button
  • Select the folder that you extracted the extension to (for example: C:\Users\YOUR_NAME\smart-browser-extension-v1.2). Press "Ok" or whatever the case may be.
  • Now a button for this extension exists in Chrome, in the toolbar, near your bookmarks. It might be buried under the "Extensions" button/menu.
    • If you want to pin the SMART extension to the toolbar (so that it's always visible and easy to reach): From chrome://extensions/ , find the SMART browser extension, press the "Details" button, find the "Pin to toolbar" toggle. Enable it.

To run it

  • Press the SMART browser extension's button. A new window will appear which contains the results.
    • The keyboard accessibility of this Chrome toolbar is unclear to me. These keyboard shortcuts might help: F10, and alt+shift+t.
  • To re-run the extension, you can press the SMART browser extension's button again twice (once to close it, once to re-open it). Or use the "Refresh" button inside of its window.
