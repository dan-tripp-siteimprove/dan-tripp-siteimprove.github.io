PICT - online

  • PICT online (not hosted by the creators of PICT).
  • Pros and cons:
    • Pro: A great way to get started with PICT. You can do a lot of useful testing with this before you run into its limitations. (i.e. the limitations of the online edition, compared to the local executable).
    • Pro: Don't need to download anything.
    • Pro: Accepts models in the same format as the local PICT .exe.
    • Con: If you want to do "not pairwise" (eg. all-triples, all-quadruples): you can't specify the "interaction strength" AKA "t" (i.e. /o cmdline arg) so use this work-around: specify a sub-model which includes all params.
    • Con: Can't use misc. other cmdline options eg. delimiters.
    • Con: Can't seed.
    • Con: Can't handle a large number of output tests. Roughly > 300 tests, or > ~ 2s to generate model. Error message: "NetworkError when attempting to fetch resource".

CAGen - online

  • CAGen is an alternative to PICT. Not covered in talk.
  • CAGen online
  • Accepts models in the ACTS format (not the PICT format). ACTS, in turn, is another generator tool, which is freely available if you send an email to the creators.
  • Pro: Can set "interaction strength" AKA "t" easily.
  • Pro: Can handle many output rows. (eg. 1 million).
  • ... Con: For 1 million output tests, the download fails (to both file and clipboard). So you can view the output in the web browser only.
  • ... Pro: For 5000 output tests: download works fine.
  • Con: Constraints can't handle integer comparisons (eg. greater-than, less-than) like PICT and ACTS can.
