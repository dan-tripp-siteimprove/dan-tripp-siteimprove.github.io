    Passes the R90 rule on a technicality? It's because of ACT rule and ARIA wording. (I don't know why they're worded this way, but they are.) The ACT rule refers to the ARIA spec, and checks (for the outer element) any roles which have 'children presentational' = true. This includes "button", "checkbox", etc. - and not "link". So all anchors as the outer element get a pass. (anchors as the inner element can still cause failures.)
